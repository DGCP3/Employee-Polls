
# Employee Polls



## Features

- Login/logout
- Create a poll
- Vote on a poll
- display leaderboard


## Run Locally

Clone the project

```bash
  https://github.com/DGCP3/employee-polls.git
```

Go to the project directory

```bash
  cd employee-polls
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
Start the mock api server

```bash
  npm run server
```
